# AI Chatbot for Government Scheme Recommendation

## 📌 Overview

AI Chatbot for Government Scheme Recommendation is an intelligent web-based application designed to help citizens identify government schemes they are eligible for. The chatbot interacts with users, collects relevant information, and provides personalized scheme recommendations based on eligibility criteria using Artificial Intelligence and Machine Learning techniques.

## 🚀 Features

* Conversational AI chatbot interface
* Personalized government scheme recommendations
* Eligibility-based scheme filtering
* Natural Language Processing (NLP) for user queries
* User-friendly and responsive design
* Quick access to scheme details, benefits, and application processes
* Real-time interaction and assistance

## 🛠️ Technologies Used

* Python
* Flask
* Machine Learning
* Natural Language Processing (NLP)
* Pandas & NumPy
* Scikit-learn
* HTML, CSS, JavaScript

## 📊 Project Workflow

1. User interacts with the chatbot.
2. Chatbot gathers user information such as age, occupation, income, category, and location.
3. Data is processed and analyzed.
4. AI/ML model determines suitable government schemes.
5. Recommended schemes are displayed along with eligibility criteria and benefits.
